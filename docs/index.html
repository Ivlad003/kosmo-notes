Built for the way you work

One small menu‑bar icon. Every feature one keystroke away.

Meeting recordings

Captures mic + system audio (and optionally screen) into rolling 5‑second segments. Crash‑safe — at most 5 seconds lost to a kernel panic, automatically recovered on next launch.

Voice notes

Press ⌘⇧N, talk, release. KosmoNotes transcribes via Whisper, picks the right template (freeform, task, journal, checklist) and saves a clean Markdown note.

Push‑to‑dictate

Hold a hotkey, speak into any text field. Whisper streams a clean transcript, optionally cleaned by Claude, then pasted into the app you're already in. No swap‑windows tax.

Hybrid transcription

Pick your engine: Deepgram, OpenAI Whisper, Gemini Audio, OpenRouter, or fully local on‑device WhisperKit. Cloud by default; local stays in the app bundle when privacy matters.

Library & semantic search

Every session lands in a SQLite‑backed Library with FTS5 + optional embeddings. Search by phrase or by meaning across months of recordings.

Share via S3

Hand‑rolled AWS Sig V4 — works with AWS, R2, B2, MinIO. Upload a session bundle and get a presigned link. No hosted viewer, no telemetry.

Chat with your meetings

Ask, and your transcripts answer back.

Open Chat. The relevant past sessions are auto‑attached via FTS — or pin them yourself. Mention "on minute 3" and the screen frame at that timestamp gets fetched from your local recording. No clouds, no copying.

  • Per‑message provider switching: Claude · GPT‑4o · OpenRouter · local Ollama
  • Live snapshots while a recording is in flight (last ~60 s)
  • Auto‑sample N evenly‑spaced video frames as vision context

Run the agent

From "summarize this" to "go do it."

Two ways to launch: hold ⌘⇧A and speak — or click the Run as agent button right inside Chat. KosmoNotes hands the conversation, attached transcripts, and your goal to an autonomous Claude loop. Watch every tool call, every result, every reply in the live console — and inject a follow‑up mid‑run.

  • Pick the model — Opus 4.7, Sonnet 4.6, Haiku 4.5 — per run
  • Strict workspace sandbox: read_file, write_file, bash (read‑only allowlist, no shell metacharacters)
  • Plug in Claude Code, Codex, or GitHub Copilot CLI as alternative backends
  • Cooperative stop, mid‑run injection, JSONL session log on disk

Privacy

Your data lives where you put it.

Sessions are flat files inside ~/Library/Application Support/KosmoNotes/recordings/<sid>/. SQLite is just a rebuildable index. No analytics, no auto‑update, no sign‑in. Use cloud transcription when you want speed, or switch to WhisperKit for fully on‑device runs.

  • Secrets in macOS Keychain — never on disk in plain text
  • Bundle ID‑pinned TCC grants — survive rebuilds
  • Open formats: audio.m4a, transcript.jsonl, summary.md, actions.json

Up and running in three minutes

No installer, no background service, no signed package — just a regular .app you drag into Applications.

  1. 1

    Download & move

    Grab KosmoNotes.zip from the latest release, double‑click to expand, drag KosmoNotes.app into /Applications.

    xattr -dr com.apple.quarantine /Applications/KosmoNotes.app

    First Gatekeeper warning is expected — KosmoNotes is unsigned. The xattr command tells macOS to trust it.

  2. 2

    Grant permissions

    On first launch the onboarding panel walks you through Microphone, Screen Recording (optional), and Accessibility (for push‑to‑dictate paste). Each one opens the right pane in System Settings.

  3. 3

    Wire up an AI provider

    Open the menu‑bar icon → Settings → AI Providers. Paste at least one API key (Anthropic for the agent and chat, OpenAI for Whisper, Deepgram for streaming transcription). Or pick WhisperKit in Transcription for a fully local pipeline.

  4. 4

    Customize hotkeys

    Defaults are ⌘⇧R Meeting · ⌘⇧N Voice Note · ⌘⇧L Library · ⌘⇧A Agent. Rebind any of them in Settings → Hotkeys.
