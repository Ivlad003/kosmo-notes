## Tag steps

Once both matrices are filled and AC-9b is recorded:

1. `xcodegen generate`
2. `xcodebuild -scheme KosmoNotes -configuration Release build` → `.app`
3. `ditto -c -k --keepParent KosmoNotes.app KosmoNotes.zip`
4. Verify zip ≤ 8 MB and unzipped ≤ 15 MB (CI gate also enforces this).
5. `git tag -a v1.0.0 -m "v1.0.0"` and push.
6. Attach `KosmoNotes.zip` + this filled checklist to the GitHub release.
